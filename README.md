This boilerplate features:
======

* Auto prefixing
* Browser syncronization
* Image compression
* Javascript linting
* Sass compiling
* CSS/JS minifying
* CSS/JS bundling


Installation:
======

npm install gulp -g

npm install

*Be sure to check if the development url in Gulpfile.js is correct. (first line of code)*

Run gulp:
======

gulp



**Happy coding!**
